기본 플랜
01

설치 방법

1
아래 링크에서 OCE_Setup.exe 파일을 다운로드합니다.
https://github.com/paperface0128/OCE/releases/latest
2
다운로드한 OCE_Setup.exe 를 실행합니다.
3
설치 마법사의 안내에 따라 진행합니다. 바탕화면 바로가기 생성 여부를 선택할 수 있습니다.
4
설치 완료 후 OCE 실행을 체크하고 마침을 누릅니다.
⚠️ Windows Defender 등 백신에서 경고가 뜰 수 있습니다. 안전한 파일이므로 추가 실행을 눌러 진행해주세요.
ℹ️ 설치 경로는 C:\Users\사용자명\AppData\Roaming\OCE 입니다. 업데이트 기능을 위해 이 경로에 설치하는 것을 권장합니다.
02

로그인 & 플랜 안내

디스코드로 로그인하기

1
툴 상단 툴바에서 디스코드 로그인 버튼을 클릭합니다.
2
브라우저가 자동으로 열리며 디스코드 로그인 페이지가 나타납니다.
3
로그인 후 권한 허용을 누르면 완료 페이지가 표시되며 툴에 자동으로 반영됩니다.
4
툴바에 현재 플랜 이름이 표시됩니다.
플랜 감정 이미지 최대 수 비고
무료 5개 기본 제공
스타터 8개 서포트 서버 상품 구매
컬렉터 무제한 서포트 서버 상품 구매
💡 플랜 업그레이드는 서포트 서버에서 상품을 구매하여 역할을 받으면 자동으로 적용됩니다. discord.gg/panette
03

캐릭터 정보 작성법

툴 상단의 새 프로젝트 버튼을 눌러 시작한 후 캐릭터 정보 탭에서 아래 항목들을 작성합니다.

기본 정보

캐릭터 ID 자동 생성됩니다. 저장 전에만 재설정 가능하며, 한 번 저장하면 변경할 수 없습니다.
이름 필수 캐릭터의 이름입니다. 반드시 입력해야 저장이 가능합니다.
설명 선택 캐릭터에 대한 간단한 설명입니다.
아이콘(이모지) 선택 캐릭터를 대표하는 이모지입니다.

스킨 목록

캐릭터의 특수 스킨을 추가합니다. 스킨 추가 버튼을 눌러 추가하세요.

normal 보상의 스킨 변경을 통해서만 바뀌는 스킨입니다. 플레이어가 특정 조건 달성 또는 이벤트를 통해 획득합니다.
season 설정한 기간(MM-DD ~ MM-DD) 동안 자동으로 강제 적용되는 스킨입니다. 예: 크리스마스 스킨 (12-24 ~ 12-26)
활성 체크박스로 스킨의 활성 여부를 설정합니다. 비활성화된 스킨은 게임에서 적용되지 않습니다.

감정 이미지 관리

캐릭터의 표정/감정 이미지를 등록합니다. 대사 스텝에서 감정을 선택하면 해당 이미지가 표시됩니다.

1
프로젝트를 먼저 저장한 후 이미지를 추가할 수 있습니다.
2
감정 이름을 입력합니다. (예: 기쁨, 슬픔, 화남, 놀람)
3
이미지 선택 & 추가 버튼으로 PNG 파일을 선택합니다.
4
특수 스킨의 감정 이미지는 스킨 감정 및 표정 할당에서 별도로 추가합니다. 이미지가 없는 감정은 기본 스킨 이미지가 자동으로 사용됩니다.
⚠️ 이미지는 반드시 PNG 형식이어야 합니다. 기본 스킨에 감정을 먼저 추가해야 특수 스킨에도 해당 감정을 할당할 수 있습니다.
04

노드 편집

노드는 대화의 흐름을 구성하는 단위입니다. 각 노드는 하나의 대화 장면이라고 생각하시면 됩니다. 왼쪽 사이드바에서 노드를 선택하고 관리할 수 있습니다.

노드 추가 & 삭제

노드 추가 사이드바 하단의 노드 추가 버튼을 클릭합니다. 이름은 한글/영문/숫자/-_ 만 사용 가능합니다.
노드 삭제 삭제할 노드를 선택한 후 노드 삭제 버튼을 클릭합니다. _START 노드는 삭제할 수 없습니다.
_START 대화가 시작되는 노드입니다. 반드시 존재해야 하며 삭제가 불가능합니다.
💡 노드 흐름 탭에서 전체 대화 구조를 시각적으로 확인할 수 있습니다. 드래그로 노드 위치 조정, 마우스 휠로 줌 인/아웃이 가능합니다.
05

대사 편집

노드를 선택하면 노드 편집 탭에서 해당 노드의 대사를 작성할 수 있습니다.

대사 스텝

대사는 여러 개의 스텝으로 구성됩니다. 각 스텝이 순서대로 출력됩니다.

화자 대사를 말하는 캐릭터를 선택합니다.
감정 해당 스텝에서 표시될 감정 이미지를 선택합니다. 캐릭터 정보에서 추가한 감정 목록이 표시됩니다.
대사 실제 출력될 대사 내용을 입력합니다.
딜레이 다음 스텝으로 넘어가기까지의 대기 시간을 설정합니다. (밀리초 단위)

다음 선택지 (버튼)

대사 출력 후 플레이어에게 보여질 선택지를 설정합니다.

버튼 텍스트 플레이어에게 보여질 선택지 문구를 입력합니다.
이동 노드 해당 선택지를 선택했을 때 이동할 노드를 지정합니다.
ℹ️ 버튼이 없으면 대화가 자동으로 종료됩니다.
06

보상 & 조건 설정

조건 설정

대사가 출력되기 위한 조건을 설정합니다. 조건을 만족하지 않으면 해당 대사는 출력되지 않습니다.

호감도 플레이어의 호감도 범위를 설정합니다. 예: 호감도 50 이상일 때만 출력
시간 특정 시간대에만 출력되도록 설정합니다. 예: 밤에만 출력
스탯 특정 스탯 수치 조건을 설정합니다.
플래그 특정 플래그가 설정된 경우에만 출력됩니다.
착용 장비 플레이어가 특정 장비를 착용한 경우에만 출력됩니다.

보상 설정

대사가 출력된 후 플레이어에게 지급되는 보상을 설정합니다.

페니 지급 또는 차감할 재화 수량입니다. 음수 입력 시 차감됩니다. 예: -100 입력 시 페니 100 차감
스킨 변경 대사 출력 후 캐릭터의 스킨을 변경합니다. 스킨 목록에 추가된 스킨만 선택 가능합니다.
아이템 지급할 아이템과 수량을 설정합니다.
장비 지급할 장비를 설정합니다.
07

저장 & 불러오기

저장 마지막으로 저장한 위치에 저장합니다. 처음 저장하는 경우 다른 이름으로 저장이 실행됩니다.
다른 이름으로 저장 새 위치를 선택하여 저장합니다. 폴더를 선택하면 캐릭터 이름으로 자동으로 폴더가 생성됩니다.
불러오기 기존에 저장한 캐릭터 폴더를 선택하여 불러옵니다.
💡 툴을 재실행하면 마지막으로 열었던 프로젝트가 자동으로 불러와집니다.
⚠️ 하단 상태바에 오류가 표시되면 저장이 불가능합니다. 오류 메시지에 마우스를 올려 내용을 확인하고 수정해주세요.
08

업데이트

툴 실행 후 2초 뒤 자동으로 업데이트를 확인합니다. 새 버전이 있을 경우 업데이트 창이 나타납니다.

지금 업데이트 업데이트를 즉시 진행합니다. 완료 후 툴이 자동으로 재시작됩니다.
나중에 업데이트를 건너뜁니다. 다음 실행 시 다시 안내됩니다.
필수 업데이트 나중에 버튼이 표시되지 않으며 반드시 업데이트해야 합니다. 봇 호환성 등 중요한 변경사항이 있을 때 적용됩니다.
09

자주 묻는 질문

설치 시 액세스 거부 경고가 뜹니다. ▼
안전한 파일입니다. 경고창에서 확인을 누르시면 정상적으로 설치가 진행됩니다.
로그인이 되지 않습니다. ▼
브라우저에서 디스코드에 로그인된 상태여야 합니다. 또한 서포트 서버에 가입되어 있는지 확인해주세요. 문제가 지속되면 서포트 서버 티켓으로 문의해주세요.
이미지 추가 버튼을 눌렀는데 아무것도 선택이 안 됩니다. ▼
프로젝트를 먼저 저장해야 이미지를 추가할 수 있습니다. 저장 버튼을 눌러 저장 후 다시 시도해주세요.
저장이 되지 않습니다. ▼
하단 상태바의 오류 메시지를 확인해주세요. 캐릭터 이름이 비어있거나 적법성 검사를 통과하지 못한 경우 저장이 불가능합니다. 오류 항목을 수정 후 다시 저장해주세요.
스킨이 스킨 목록에 표시되지 않습니다. ▼
불러오기 시 이미지 폴더를 자동으로 스캔하여 스킨을 추가합니다. 저장 버튼을 눌러 character.json에 반영해주세요.
감정 이미지 수 제한에 걸렸습니다. ▼
현재 플랜에서 허용된 감정 이미지 수를 초과했습니다. 서포트 서버에서 상위 플랜을 구매하시면 더 많은 감정 이미지를 추가할 수 있습니다. discord.gg/panette
업데이트 후 툴이 실행되지 않습니다. ▼
설치 폴더(C:\Users\사용자명\AppData\Roaming\OCE)에서 OCE.exe를 직접 실행해보세요. 문제가 지속되면 재설치 후 서포트 서버에 문의해주세요.

해결되지 않는 문제는 서포트 서버 티켓으로 문의해주세요.
